MeTVe V2 is a virtual interactive cable-style television platform where online users can create and manage professional channels, upload media, run live or scheduled playout, and engage viewers from a PC browser with cloud-like workflows.

  • Create, save and manage channels with stable API + socket fallback behavior.
  • Upload and organize shows, movies, commercials, songs, promos, graphics, images and GIF media.
  • Run SD/HD/UHD playout with scheduling, fillers, and emergency override simulation.
  • Use creator studio controls for monetization, audience engagement, and operator actions across Slide TV, Video TV, Radio TV, VIP, Ringtone TV, Chat, Games, Other, and All channels.
